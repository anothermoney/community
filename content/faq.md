# Frequently asked questions

### Is another.money free?

The app is in alpha and free to try on the web. Core personal-finance use is the
heart of the app; some advanced features (like multi-device collaboration) are
premium. Pricing will be announced before general availability.

### Do I need an account to use it?

No. There's no sign-up. You create a local book with a password and start using
it immediately. An account is only relevant if/when you opt into cloud features.

### Where is my data stored?

In an encrypted book file on your device. On the web that's in your browser's
local storage; on mobile it's in the app's storage. Use **Back up** to keep a
portable copy.

### What if I forget my book password?

There's no reset — the password encrypts the data and there's no backdoor. If
you lose it, the book can't be opened. Keep your password safe and keep a backup.

### Can I use it on my phone and computer together?

Yes — see [Collaboration](/collaboration). Multiple devices can share one book,
end-to-end encrypted, and sync automatically.

### Can I import from my old app or bank?

Yes. another.money imports **QIF, OFX, CSV, and MT940**. It's also a faithful
rebuild of the 2014 *anMoney* data model, so its concepts will feel familiar to
former users.

### Does it support multiple currencies?

Yes. Accounts can be in different currencies, and cross-currency amounts convert
using the transaction's own date so historical figures stay accurate.

### Is there an iOS / Android app?

The web app is live in alpha. Native iOS and Android builds are on the way.

### My sync says it's connected but changes aren't showing up.

Make sure both devices use the same book password and are online, and try
**Sync now**. If it persists, please report it (below) with what you were doing
on each device.

### How do I report a bug or suggest a feature?

The app has a built-in bug-report option, and this site's content lives in a
public repository you can open issues and pull requests against. Friendly,
specific reports help a lot.

### Is another.money open source?

The community content (this site, guides, FAQ) is open for contributions. See the
repository's `CONTRIBUTING` guide.
